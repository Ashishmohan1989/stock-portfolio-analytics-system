# 📊 Stock Portfolio Management & Analytics System

## 🚀 Project Overview

This project is a **data-driven stock portfolio management system** built using **Python, MySQL, and Pandas**. It simulates real-world portfolio tracking by storing transactions, calculating holdings dynamically, and generating analytical insights.

---

## 🎯 Problem Statement

Managing stock investments manually (Excel or notes) leads to:

* Errors in profit/loss calculation
* Difficulty tracking transactions
* Lack of structured data

This system solves these problems using a **database-driven analytics approach**.

---

## 🧠 Key Features

* 📌 Relational database design (Users, Stocks, Trades, Holdings, Watchlist)
* 📌 Trade-based portfolio tracking (BUY/SELL simulation)
* 📌 Dynamic holdings calculation (derived from transactions)
* 📌 Portfolio analytics using Pandas
* 📌 CLI-based interaction for data retrieval
* 📌 Export-ready data for reporting

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy**
* **MySQL**
* **SQLAlchemy**
* **CLI (Command Line Interface)**

 ⚙️ How It Works

1. Stock and user data is stored in MySQL
2. Trades (BUY/SELL) are recorded
3. Holdings are dynamically calculated from transactions
4. Pandas is used to generate analytics and summaries
5. CLI allows users to interact with the system
